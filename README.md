# Rewire

> Helping users recognize when they've fallen into a dopamine loop and giving them practical opportunities to regain control of their attention.

---

## Overview

**Rewire** is an Android application designed to reduce compulsive social media usage by interrupting unhealthy scrolling patterns with intentional interventions.

Unlike traditional screen-time blockers, Rewire focuses on **awareness rather than restriction**. The app detects prolonged usage patterns and introduces lightweight interventions that help users consciously decide whether to continue using the application.
